# Earnings Calendar — Calendário Macro

## Quando usar
Sempre que mencionar datas de eventos macro: CPI, FOMC, COPOM, earnings de empresas, vencimento de opções B3.

## REGRA CRÍTICA — Datas são FATOS
NUNCA invente ou estime datas de divulgação macro.
Se não souber a data exata, diga:
"Não tenho a data confirmada. Consulte: https://economic-calendar.tradingview.com"

## Calendário de referência (atualizar mensalmente)

### Vencimento de Opções B3
- Opções mensais: 3ª segunda-feira de cada mês
- Opções semanais: toda segunda-feira
- Próximos vencimentos mensais 2026: 18/05, 15/06, 20/07, 17/08, 21/09

### Eventos Macro Brasil
- COPOM: reuniões a cada 45 dias (datas no site BCB)
- IPCA: divulgado pelo IBGE ~dia 9 de cada mês
- PIB: trimestral (IBGE)
- Fonte oficial: https://www.bcb.gov.br/publicacoes/agendabcb

### Eventos Macro EUA
- CPI: divulgado ~dia 10-15 de cada mês pelo BLS
- FOMC: 8 reuniões por ano (calendário em federalreserve.gov)
- Payroll (NFP): primeira sexta de cada mês
- Fonte oficial: https://www.federalreserve.gov/monetarypolicy/fomccalendars.htm

### Como verificar data atual
Se o usuário perguntar sobre um evento específico, responda:
"Para a data exata, consulte: https://economic-calendar.tradingview.com ou https://www.bcb.gov.br"
